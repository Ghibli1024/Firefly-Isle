# Firefly-Isle PRD (Product Requirements Document)

**产品名称：** 一页萤屿 (Firefly-Isle)
**定位：** One-page oncology treatment timeline & record builder
**Slogan：** Keep it running, make it helpful.

---

## 1. 背景与问题

本项目源于癌症患者/患者家属的真实需求：

- 晚期癌症患者因频繁的复发进展，往往面临**多线治疗方案**。
- 患者/家属整理病历时，因**信息过载**而感到无助。
- 异地就医沟通时，因患者数量过多，分配给个人的时间过短，无法与医生做好详尽沟通。

**核心目标：** 帮助晚期癌症患者/家属高效管理治疗方案，在有限的就诊时间内与医生完成清晰沟通。

---

## 3. MVP 核心功能（P0）

### 3.1 自然语言输入 → 结构化信息提取

用户打开界面时，看到一个**干净的界面**，只有一个提示语，引导用户说出病情。

#### 数据模型

**基本信息（均为可选）：**
- 性别、年龄、身高、体重
- 肿瘤类型、确诊时间、临床分期

**患者分类与表格结构：**

所有患者使用同一套表格模板，AI 根据患者情况自动填充对应区块。

```
情况 1 — 非晚期患者
  初发区块
    - 触发时间（可选）
    - 治疗方案（可选）
    - 免疫组化（可选）
    - 基因检测（可选）
  （无治疗线区块）

情况 2 — 确诊即晚期（初发 = 一线）
  治疗线区块（可无限添加）
    一线 / 二线 / 三线...
      - 时间段
      - 用药方案
      - 穿刺（可选）
      - 免疫组化（可选）
      - 基因检测（可选）

情况 3 — 复发晚期（有初发记录 + 复发后多线）
  初发区块
    - 触发时间（可选）
    - 治疗方案（可选）
    - 免疫组化（可选）
    - 基因检测（可选）
  治疗线区块（可无限添加）
    一线 / 二线 / 三线...
      - 时间段
      - 用药方案
      - 穿刺（可选）
      - 免疫组化（可选）
      - 基因检测（可选）
```

**核心约束：**
- 免疫组化、基因检测**跟随每线治疗**，不在表格末尾统一显示
- 所有字段均为可选，AI 只提取用户实际提供的信息
- 未填字段在表格中留白，预览页高亮提示补充

### 3.2 一键生成极简表格

- 将提取的结构化信息转换为**一页极简表格**（参见附件示例）
- 表格设计以一页纸可呈现为目标，便于就诊时直接递给医生

### 3.3 二次编辑

- 生成后支持**手动编辑**表格内容
- 支持在**输入框中对话式修改**（通过 AI 交互修正）

### 3.4 导出与下载

- 导出格式：PDF、图片

## 4. 后续开发功能（P1）— 前期架构需预留

### 4.1 血液检查报告 AI 识别

- 识别血常规、血生化、肿瘤标志物报告
- 自动绘制趋势表格
- 将**异常且持续增高**的指标标记高亮

### 4.2 拍照上传病历

- 支持用户**拍照上传**纸质病历
- AI 自动识别并提取关键信息，填入结构化表格

---

## 5. 后续可选优化（P2）

### 5.1 功能优化

- **加密分享：** 治疗方案一键转发，需个人授权码加密
- **健康查询：** 接入 MSD 手册 (https://www.msdmanuals.cn/home)
- **AI 助理：** 分析治疗方案及血液指标
- **全平台适配：** Windows、Mac、Android、iOS，酌情适配小程序、鸿蒙
- **多语种：** 至少支持中文和英文
- **扩展能力：** 结合 OpenClaw、WebChrome 等技术

### 5.2 美工与体验优化

- **背景音乐播放器：** 《Just One Dance》
- **欢迎页：** 拥抱动画 + 鼓励语，下置按钮进入主界面（可考虑镜面、玻璃等设计元素）
- **许愿墙：** 用户匿名许愿，给人以希望
- **每日鼓励：** 用户提交鼓励语，附带署名
